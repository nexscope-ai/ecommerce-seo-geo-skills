# E-commerce GEO Optimizer v1.0.0

## What This Skill Does

Analyze and optimize e-commerce pages for **Generative Engine Optimization (GEO)** — making product pages, listings, and store content visible, citable, and recommendable by AI search engines including ChatGPT, Claude, Perplexity, Gemini, and Google AI Overview.

GEO is distinct from traditional SEO:
- **SEO** = get found in Google search results
- **GEO** = get recommended when shoppers ask AI for buying advice

---

## When to Use This Skill

- Product page not appearing in AI-generated buying recommendations
- Store getting traffic from Google but missing AI search visibility
- Launching a new product and want AI-first visibility strategy
- Competitor appears in ChatGPT/Perplexity recommendations but you don't
- Optimizing listings for voice search and conversational queries
- Preparing A+ content or store pages for AI citation readiness

## Core GEO Framework for E-commerce

### The 8 GEO Pillars

| Pillar | Focus | Impact |
|--------|-------|--------|
| **1. Crawl & SEO Readiness** | Title, meta, H1, canonical, robots, sitemap | Foundation |
| **2. Image SEO** | Alt text, filenames, WebP, product image metadata | High |
| **3. Structured Data** | Product, Offer, Review, FAQ, Organization schema | Very High |
| **4. AI Brand Recognition** | Brand, category, materials, colors, use cases clearly stated | Very High |
| **5. Buyer Intent Coverage** | Use cases, pain points, comparisons, budget signals | High |
| **6. Trust & Conversion Proof** | Reviews, ratings, warranty, shipping, returns, certifications | High |
| **7. AI Citation Readiness** | FAQ blocks, concise answer sections, source-friendly claims | Very High |
| **8. Conversational Content** | Natural language answers to buyer questions | High |

## Step-by-Step GEO Analysis Workflow

### Step 1 — Understand the Page

Before analyzing, identify:
- Page type: product page / category page homepage / marketplace listing
- Platform: Shopify / WooCommerce / Amazon / DTC / Other
- Target AI engines: ChatGPT / Perplexity / Gemini / Claude / Google AI Overview
- Primary product category and target buyer persona
- Current SEO status if known

### Step 2 — GEO Audit by Pillar

Run through each of the 8 pillars and assess:

**Pillar 1: Crawl & SEO Readiness**
```
✅ Check: Is the page indexable? (no noindex, no robots block)
✅ Check: Title tag — does it include primary keyword + brand?
✅ Check: Meta description — does it answer "what is this product?"
✅ Check: H1 tag — matches title intent?
✅ Check: Canonical tag — is it self-referencing or pointing correctly?
✅ Check: Sitemap inclusion
✅ Check: Page load speed (<3s target)
```

**Pillar 2: Image SEO & Product Understanding**
```
✅ Check: All product images have descriptive alt text
✅ Check: Filenames are descriptive (not IMG_001.jpg)
✅ Check: WebP or AVIF format used for performance
✅ Check: Product images show multiple angles
✅ Check: Lifestyle images showing product in use
✅ Check: Image dimensions appropriate for platform
```

**Pillar 3: Structured Data & Rich Signals**
```
✅ Check: Product schema (name, description, brand, sku, image)
✅ Check: Offer schema (price, priceCurrency, availability)
✅ Check: AggregateRating schema (reviews + ratings)
✅ Check: Organization schema (brand identity)
✅ Check: BreadcrumbList schema (navigation context)
✅ Check: FAQPage schema (if FAQ section exists)
✅ Check: No schema errors in Google Rich Results Test
```

**Pillar 4: AI Brand Recognition**
```
✅ Check: Brand name mentioned prominently in first 100 words
✅ Check: Product category stated clearly
✅ Check: Materials/ingredients/specifications listed explicitly
✅ Check: Colors, sizes, variants clearly described
✅ Check: Use cases stated in natural language
✅ Check: Offer context (price range, value proposition)
```

**Pillar 5: Buyer Intent Coverage**
```
✅ Check: Content addresses "best for" queries
✅ Check: Comparison content (vs competitors / vs alternatives)
✅ Check: Budget-fit signals ("affordable", "premium", price context)
✅ Check: Use-case specificity ("perfect for camping", "ideal for sensitive skin")
✅ Check: Gifting scenarios addressed if relevant
✅ Check: Pain points and solutions explicitly stated
```

**Pillar 6: Trust & Conversion Proof**
```
✅ Check: Review count and average rating visible
✅ Check: Warranty / guarantee information present
✅ Check: Shipping speed and return policy stated
✅ Check: Certifications and compliance badges (FDA, CE, etc.)
✅ Check: User-generated content / social proof
✅ Check: Number of customers / units sold if available
```

**Pillar 7: AI Citation Readiness**
```
✅ Check: FAQ section with direct Q&A format
✅ Check: Short answer-ready paragraphs (2-3 sentences)
✅ Check: Claims are specific and citable ("reduces pain by 40%")
✅ Check: Content can be summarized without losing meaning
✅ Check: No heavy reliance on images to convey key info
✅ Check: Key facts repeated in multiple formats (bullets + prose)
```

**Pillar 8: Conversational Content**
```
✅ Check: Content written in natural, conversational language
✅ Check: Addresses "who is this for?" directly
✅ Check: Addresses "why buy this vs alternatives?" directly
✅ Check: Long-tail question phrases covered in headings
✅ Check: Buyer journey stages covered (awareness → decision)
```

### Step 3 — Scoring

Score each pillar:
- ✅ **Strong** (3 pts) — Most checks pass
- ⚠️ **Partial** (2 pts) — Some checks pass, obvious gaps
- ❌ **Weak** (1 pt) — Most checks fail, major work needed

**Total GEO Score: X/24**

| Score | Rating | Action |
|-------|--------|--------|
| 20-24 | 🟢 Excellent | Maintain and monitor |
| 15-19 | 🟡 Good | Target weak pillars |
| 10-14 | 🟠 Needs Work | Priority optimization needed |
| Below 10 | 🔴 Poor | Full GEO overhaul required |

### Step 4 — Prioritized Fix Plan

Organize fixes by impact and effort:

**🔴 High Impact / Low Effort (Do First)**
- Add FAQ schema markup
- Add Product schema if missing
- Add clear brand name in opening paragraph
- Optimize alt text on all product images

**🟡 High Impact / Medium Effort (Do Next)**
- Write FAQ content section targeting buyer questions
- Add structured use-case and comparison content
- Improve product description with buyer intent language
- Add AggregateRating schema

**🟢 Medium Impact / Higher Effort (Plan For)**
- Rewrite product description for conversational voice
- Add lifestyle imagery with proper metadata
- Create comparison page or section
- Build trust section with certifications and guarantees

## Platform-Specific GEO Notes

### Amazon Listings
- Amazon controls most technical SEO — focus on content signals
- Product title: include primary keyword + key feature + brand
- Bullet points: lead with benefit, include buyer language
- A+ Content: use FAQ modules and comparison charts
- Backend keywords: include voice search phrases and long-tail questions

### Shopify Stores
- Full control over schema — implement Product + Offer + Review schema
- Blog content supports GEO — create buyer guide content
- Collection pages need unique descriptions (not just product lists)
- Use apps for schema automation (JSON-LD for SEO, Smart SEO)

### WooCommerce
- Yoast SEO or Rank Math for schema automation
- Product schema generated automatically — verify completeness
- Add FAQ blocks using page builders with schema support
- Enable reviews and ensure they display with structured markup

### Marketplace Listings (eBay, Etsy, Walmart)
- Limited technical control — focus on content signals
- Title optimization: buyer language over keyword stuffing
- Description: use natural language, include FAQs in description body
- Image quality and quantity signal product authority to AI

## GEO vs SEO Quick Reference

| Factor | SEO Focus | GEO Focus |
|--------|-----------|-----------|
| **Content format** | Keyword density | Answer-ready paragraphs |
| **Structure** | Header hierarchy | FAQ + schema markup |
| **Trust signals** | Backlinks | Reviews, certifications, proof |
| **Language** | Keyword-optimized | Conversational, buyer-language |
| **Images** | Alt text | Rich metadata + context |
| **Authority** | Domain authority | Citable, specific claims |
| **Measurement** | Rankings, traffic | AI citation frequency |

---

## Common GEO Mistakes to Avoid

❌ **Keyword stuffing** — AI reads for context and clarity, not density

❌ **Thin product descriptions** — One paragraph is not enough for AI to cite you

❌ **Missing schema** — Without structured data, AI can't extract product facts reliably

❌ **Vague claims** — "Great quality" is not citable; "Rated 4.8/5 by 2,000+ customers" is

❌ **No FAQ content** — FAQ format is how AI pulls direct answers

❌ **Image-only specs** — If key info is only in images, AI can't read it

❌ **No brand clarity** — If AI can't identify your brand in the first 100 words, you lose

❌ **Ignoring mobile** — AI search is heavily mobile-driven
